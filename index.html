SFNG

(Sci Fi Name Generator)

Hello, Esteemed Internet Wanderer, and welcome to the Sci Fi Name Generator!

It pulls random letters, numbers, and one special character from a few arrays and concatonates them into a word on the DOM. The user also has the option to remove the last character or deleting the entire word and starting over.

If you'd like to know some more about it, not that's there's much more to know, you can check out the repo here.
